# 숨고 데모용 최종 수정 가이드

> 데모 목적: 숨고 포트폴리오로 사용 → 고객이 보고 "이런 걸 만들어 달라"고 요청하게 만드는 것
> 기능이 완벽할 필요 없음. **보여지는 퀄리티**와 **신뢰감**이 핵심.

---

## 핵심 원칙

1. **고객 눈에 보이는 것만 깔끔하게** — 개발자 전용 메뉴, 테스트 페이지는 숨긴다
2. **실제 업체처럼 보이게** — 업체명, 주소, 전화번호를 그럴듯한 가상 정보로 채운다
3. **CTA(행동 유도)를 명확하게** — "문의하기", "예약하기" 버튼이 눈에 잘 띄어야 한다
4. **깨지거나 빈 화면이 없어야** — 데이터 없는 빈 상태가 노출되면 신뢰 하락

---

|---------|------|------|

# 수정 포인트 상세

---

## [1] 헤더 메뉴 — 🔴 필수

### 현재 문제
현재 상단 메뉴 구성:
```
Home | 회원 | 메인 선택 | 관리자
```

- **"메인 선택"** 은 개발용 테마 스위처. 고객에게 노출되면 혼란
- **"관리자"** 는 메가메뉴에 "업로드 테스트" 같은 개발 항목 포함
- 실제 피트니스 사이트처럼 보이지 않음

### 목표 구성
```
GymPT 로고  |  홈  트레이너  프로그램  예약  [로그인]  [PT 예약 →]
```

### 변경 사항
- `menuData.js`에서 `theme` 항목 제거
- `admin` 메뉴에서 "업로드 테스트" 항목 제거
- 네비게이션을 고객 관점으로 재구성
  - `홈` → `/`
  - `트레이너` → `/trainers`
  - `프로그램·패스` → `/passes`
  - `PT 예약` → `/reserve`
  - (관리자는 숨기고, 푸터 하단에 작게 링크 또는 별도 `/admin` 직접 접속)

---

## [2] 업체 정보 — 🔴 필수

### 현재 문제
`HomePageV6.vue`에 하드코딩된 기본값:
```
업체명: 짐 PT
주소: 서울시 강남구 테헤란로 123
전화: 02-0000-0000
```
전화번호 `0000-0000` 이 노출되면 미완성처럼 보임

### 변경 사항
아래 가상 정보로 교체 (또는 실제 본인 정보 사용):
```
업체명: GymPT 강남점
주소: 서울특별시 강남구 테헤란로 427
전화: 02-1234-5678
```
히어로 타이틀, 서브타이틀도 좀 더 임팩트 있게 다듬기

---

## [3] 푸터 정보 정리 — 🔴 필수

### 현재 문제
`HomeSiteFooter.vue`의 내용이 확인 필요.
주소·전화·이메일·사업자번호 등이 비어있거나 플레이스홀더면 신뢰도 하락.

### 변경 사항
- 업체명, 주소, 전화 기재
- SNS 링크 (인스타그램 등) 채우기
- 저작권 문구: `© 2025 GymPT. All rights reserved.`
- (선택) 관리자 페이지 링크를 푸터 최하단에 작게 추가

---

## [4] 데모 안내 배너 — 🟡 권장

### 이유
고객이 실제 운영 중인 사이트와 데모를 혼동할 수 있음.
반대로, "이것은 데모입니다"를 명시하면 신뢰감 오히려 상승.

### 추가 위치
헤더 바로 위 (또는 헤더 안) 상단 고정 띠 배너

### 내용 예시
```
🎯 이 사이트는 포트폴리오 데모입니다. 제작 문의: 카카오톡 [오픈채팅 링크] 또는 숨고 프로필 →
```

---

## [5] 페이지 타이틀 & 파비콘 — 🟡 권장

### 현재 문제
`router/index.js`에서 타이틀이 `· Gym PT`로 설정됨. 파비콘이 기본 Vite 아이콘일 가능성 높음.

### 변경 사항
- `index.html` 파비콘을 간단한 아령/불꽃 이모지 파비콘으로 교체
  (또는 `/public/favicon.ico` 교체)
- 기본 페이지 타이틀: `GymPT — 1:1 퍼스널 트레이닝 예약 시스템`

---

## [6] 관리자 메뉴 노출 범위 — 🟡 권장

### 현재 문제
헤더에서 관리자 메가메뉴가 모든 방문자에게 보임.
"업로드 테스트" 같은 항목이 노출되면 미완성 느낌.

### 변경 방향
- 헤더에서 관리자 메뉴 완전히 제거
- 대신 푸터 하단에 `관리자 로그인` 링크 작게 추가
- 또는 헤더 우측에 아이콘 버튼 하나로만 축소

---

## [7] OG 메타태그 — 🟢 선택

### 이유
숨고에 데모 링크를 공유할 때, 카카오톡·SNS에서 미리보기가 예쁘게 나오면 전문성 UP

### `index.html`에 추가할 내용
```html
<meta property="og:title" content="GymPT — 1:1 퍼스널 트레이닝 예약 시스템" />
<meta property="og:description" content="트레이너 예약부터 패스 관리, 관리자 대시보드까지 피트니스 풀패키지 데모" />
<meta property="og:image" content="[히어로 이미지 URL]" />
<meta property="og:type" content="website" />
```

### 프롬프트 7 — 헤더에서 메가메뉴 제거 후 심플 네비로 변경

