# GymPT 메인화면 (HomePageV6) 분석서

작성일: 2026-03-26

---

## 1. 개요

`HomePageV6.vue`는 GymPT 예약 데모의 메인 랜딩 페이지다.
Vue 3 Composition API 기반이며, 전체 섹션을 독립 컴포넌트로 분리한 조립형 구조를 채택했다.
스크롤 애니메이션 directive(`v-scroll-animate`), Tailwind CSS, scoped CSS 토큰을 병행 사용한다.

## 3. 섹션별 상세 분석

### 3-1. HomeHeroV6 — 히어로 슬라이더

**구조**
- 높이 `90vh` 풀스크린 섹션
- 배경 이미지 4장 자동 슬라이드 (4초 간격, `setInterval`)
- 좌측 상단: 타이틀 (`Playfair Display` 서리프 폰트, `clamp(2rem, 4vw, 3.5rem)`)
- 좌측 하단: 추천 프로그램 플로팅 카드 (glassmorphism, `backdrop-filter: blur(8px)`)
- 우측 하단: 서브타이틀 + dot indicator

**인터랙션**
- dot 클릭 시 슬라이드 이동 + 타이머 리셋
- 활성 dot은 8px → 32px 가로 확장 + 흰색 프로그레스 fill 애니메이션 (4000ms)
- 이미지 전환: `opacity` 0→1, `transition: 0.8s ease`

**데이터**
- `heroSlides`: hero 이미지 + aboutSections 이미지 2장 + 추가 이미지로 4장 구성
- `featuredProgram`: 프로그램 목록 첫 번째 항목 자동 선택
- API(`getStoreInfo`)에서 hero, aboutSections, programs 오버라이드 가능

**반응형**
- 640px 이하: 플로팅 프로그램 카드 숨김, 타이틀/푸터 여백 축소

---

### 3-2. AboutHeroSection — 브랜드 소개

**구조**
- 3열 CSS Grid: 왼쪽 세로 레일(56px) / 중앙 본문 / 오른쪽 통계 패널(220px)
- 왼쪽: `ABOUT US` 세로 타이포그래피 레일
- 중앙: 태그라인 → 대형 헤딩(`하나의 / 공동체입니다.`) → 설명 → CTA 버튼 2개
- 오른쪽: 통계 4개 (500+ 회원 / 98% 만족도 / 10+ 트레이너 / 5년 경력)

**디자인 특징**
- 라임 포인트 컬러 `#d4f24a` — 헤딩 하이라이트 밑줄, 태그 라인 등 반복 사용
- 헤딩 강조 단어에 기울어진 라임색 배경 pseudo-element
- 우측 대각 라임 그라디언트 배경 장식

**반응형**
- 900px 이하: 통계 패널이 하단으로 이동, 2×2 그리드로 변환
- 600px 이하: 세로 레일 숨김, 단일 컬럼

**다크모드** — `prefers-color-scheme: dark` 대응 완료

### 3-3. AnalyticsWorkoutSection — 분석 기반 운동

**구조**
- 좌 60% / 우 40% 레이아웃
- 좌: 트레이닝 이미지 + 플로팅 도넛 차트 카드 (`-rotate-6` 기울기)
- 우: 대형 헤딩 + 설명 텍스트

**기술적 특징**
- SVG 도넛 차트 직접 구현 (`stroke-dasharray`로 진행률 표현, 48.2%)
- `linearGradient`로 보라→연보라 그라디언트 stroke
- `useId()`로 SVG gradient ID 충돌 방지
- 이미지 prop 없으면 Unsplash 기본 이미지 사용

### 3-4. BreakLimitsSection — 이미지 + 카피

**구조**

- 우 상단: 이미지 + 컬러 스트라이프 장식 (노랑/주황/보라/연보라, 우측 상단)
- 우 하단: 헤딩 + 설명 + CTA 버튼

### 3-5. FeaturesSection — 3대 특징

**구조**
- 3열 카드 그리드 (960px 이하: 단일 컬럼 수평 레이아웃)
- 각 카드: 배경 대형 번호(01/02/03) + 아이콘 + 제목 + 설명 + 하단 "자세히 알아보기"

**애니메이션**
- 아이콘 주위 링 pulse 애니메이션 (3초 주기, 1초 딜레이 차이로 2개 교차)
- 아이콘 SVG float 애니메이션 (4초 주기, translateY + rotate)
- 카드 hover 시 `translateY(-6px)` + 그림자 + 링 pulse 가속(1.5초)

**아이콘 3종**
- `warning`: 방패 + 느낌표 → "소수 정예 수업" (주황)
- `clock`: 시계 + 번개 → "24시간 케어" (파랑)
- `check`: 원형 체크 → "진짜 경험의 코칭" (라임)

## 4. 스크롤 애니메이션 시스템

### 구조
```
directives/scrollAnimate.js     — Vue directive 등록
composables/useScrollAnimation.js — IntersectionObserver 유틸
assets/scrollAnimate.css        — 애니메이션 CSS 클래스
```

### 동작 방식
1. `v-scroll-animate` directive가 요소에 `.scroll-animate` 클래스 추가
2. `IntersectionObserver`로 뷰포트 진입 감지
3. 진입 시 `.scroll-animate--in` 클래스 추가 → CSS transition 실행
4. `data-animate-on-load="true"` 속성 시 페이지 로드 즉시 실행 (히어로 배경)
5. `prefers-reduced-motion` 감지 → 모션 없이 즉시 표시

### 지원 애니메이션 타입
| 타입 | 효과 |
|------|------|
| `fade-up` | 아래서 위로 올라오며 나타남 |
| `fade-right` | 왼쪽에서 오른쪽으로 슬라이드 |
| `fade-left` | 오른쪽에서 왼쪽으로 슬라이드 |
| `fade-down` | 위에서 아래로 내려오며 나타남 |
| `zoom-in` | 작게 시작해서 확대 |
| `zoom-out` | 크게 시작해서 축소 (히어로 배경) |

### 딜레이 시스템
- `data-delay="숫자(ms)"` 속성으로 지연 설정
- 같은 섹션 내 요소들에 `0, 100, 200, 400ms` 등 순차 지연 적용

## 5. 데이터 흐름

```
HomePageV6.vue (부모)
│
├── onMounted → getStoreInfo(API)
│     ├── storeName, storeAddress, storePhoneRaw
│     ├── hero (title, subtitle, image)
│     ├── aboutSections (2개)
│     └── programs
│
├── heroSlides (computed) ← hero.image + aboutSections 이미지 + 추가 1장
├── featuredProgram (computed) ← programs[0]
├── storePhoneDisplay (computed) ← formatKoreanPhoneDisplayOrDash()
├── googleMapEmbedUrl (computed) ← storeAddress
└── googleMapLinkUrl (computed) ← storeAddress
```

API 실패 시 모든 값은 하드코딩된 기본값 유지 (graceful degradation).

## 6. 레이아웃 리듬 시스템

`HomePageV6.vue` scoped 스타일에서 `.home-v6-rhythm` 클래스로 섹션 간격을 통일한다.

```css
--home-section-py: clamp(3rem, 5vw, 5rem);
```

- 히어로(`.hero-v6`)와 스토리 캐러셀(`.v6-story-carousel`)은 제외 — 풀스크린 레이아웃 유지
- 나머지 `section`, `footer`에 일괄 `padding-top/bottom` 적용 (`!important`)
- 뷰포트 너비에 따라 3rem~5rem 사이 유동값

## 8. 접근성

- 모든 섹션에 `aria-labelledby` 적용
- 장식 요소에 `aria-hidden="true"`
- 슬라이더 dot에 `role="tablist"`, `aria-label`
- 트레이너 목록에 `role="list"`
- `prefers-reduced-motion` 존중 — 애니메이션 비활성화 분기 처리
- 이미지에 `alt` 텍스트 일관 적용
