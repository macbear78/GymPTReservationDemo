## 1. 사이트 소개

이 시스템은 헬스장 회원이 **온라인으로 직접 PT를 예약**하고, 관리자(직원)가 **예약 현황을 한눈에 관리**할 수 있는 웹 서비스입니다.

**주요 기능 요약**

| 대상 | 기능 |
|------|------|
| 회원 | 트레이너 선택 → 날짜/시간 선택 → PT 예약 |
| 회원 | 전화번호로 내 예약 내역 조회 |
| 관리자 | 전체 예약 조회, 상태 변경, 취소 처리 |
| 관리자 | 트레이너 등록 및 프로필 관리 |

---

## 2. 회원이 예약하는 방법

회원은 별도의 회원가입이나 로그인 없이 예약할 수 있습니다.

### 순서

**① 홈 화면에서 "PT 예약하기" 버튼 클릭**

---

**② 트레이너 선택**

등록된 트레이너 목록이 표시됩니다. 원하는 트레이너를 선택하세요.

---

**③ 날짜 선택**

달력에서 원하는 날짜를 선택합니다.

---

**④ 시간 선택**

선택한 트레이너와 날짜에서 **예약 가능한 시간대만 활성화**되어 표시됩니다.
이미 예약된 시간은 회색으로 비활성화됩니다.

> 운영 시간: 06:00 / 07:00 / 08:00 / 09:00 / 18:00 / 19:00 / 20:00

---

**⑤ PT 유형 선택**

- **30분** 수업
- **60분** 수업

---

**⑥ 예약자 정보 입력**

- 이름
- 전화번호 (예: 01012345678)

---

**⑦ "예약하기" 버튼 클릭**

예약이 완료되면 예약 요약 화면으로 이동하며, 예약 정보(트레이너, 날짜, 시간, PT 유형, 이름, 연락처)를 확인할 수 있습니다.

---

## 3. 예약 내역 조회하기 (회원)

회원은 **전화번호**를 입력해 본인의 예약 목록을 확인할 수 있습니다.

1. 상단 메뉴에서 **"내 예약"** 클릭
2. 예약 시 입력한 전화번호 입력
3. 조회 버튼 클릭 → 예약 목록 표시

각 예약에는 다음 정보가 표시됩니다:
- 예약 날짜 및 시간
- 담당 트레이너
- PT 유형 (30분 / 60분)
- 현재 상태 (확정 / 완료 / 취소)

## 4. 관리자 페이지 사용법

관리자 페이지 주소: `사이트주소/admin`

> 현재 별도의 관리자 로그인이 없습니다. 직원 외 외부에 주소가 노출되지 않도록 주의하세요.

---

### 4.1 예약 현황 보기

관리자 페이지 상단에 **4가지 통계 카드**가 표시됩니다.

| 카드 | 내용 |
|------|------|
| 오늘 예약 | 오늘 확정된 예약 수 |
| 이번 주 예약 | 이번 주 확정된 예약 수 |
| 총 예약 | 전체 예약 수 (취소 포함) |
| 취소 예약 | 취소 처리된 예약 수 |

그 아래 **예약 목록 테이블**에서 모든 예약의 상세 내역을 확인할 수 있습니다.

---

### 4.2 예약 검색 및 필터

| 기능 | 사용 방법 |
|------|----------|
| 이름 검색 | 이름 입력란에 회원 이름 입력 |
| 전화번호 검색 | 전화번호 입력란에 번호 입력 (일부만 입력해도 됩니다) |
| 트레이너 필터 | 드롭다운에서 특정 트레이너 선택 |

검색 조건은 동시에 적용됩니다. 예를 들어 트레이너를 선택한 상태에서 이름도 검색하면 해당 트레이너의 예약 중 이름이 일치하는 것만 표시됩니다.

---

### 4.3 예약 상태 변경

예약 목록에서 각 예약 행의 **상태 드롭다운**을 통해 상태를 변경할 수 있습니다.

| 상태 | 설명 |
|------|------|
| **Confirmed (확정)** | 예약이 접수된 기본 상태 |
| **Completed (완료)** | PT 수업이 완료된 경우 |
| **Cancelled (취소)** | 예약이 취소된 경우 |
| **No-show (노쇼)** | 회원이 나타나지 않은 경우 |

상태를 변경하면 즉시 저장됩니다.

---

### 4.4 예약 취소하기

예약 행의 오른쪽 **"취소"** 버튼을 클릭하면 확인 팝업이 표시됩니다.
"확인"을 누르면 해당 예약이 삭제됩니다.

> **주의**: 취소된 예약은 복구할 수 없습니다.

## 5. 트레이너 등록하기

트레이너 등록 페이지 주소: `사이트주소/admin/trainers`

새 트레이너를 등록할 때 아래 정보를 입력합니다.

| 항목 | 예시 | 필수 여부 |
|------|------|----------|
| 트레이너 ID | `kim` (영문, 숫자, -, _ 만 가능) | 필수 |
| 이름 | 김트레이너 | 필수 |
| 전문 분야 | 근력 트레이닝 | 필수 |
| 경력 | 5년 경력 | 필수 |
| 가격(원/시간) | 60000 | 필수 |
| 프로필 이미지 | 사진 파일 선택 후 업로드 | 필수 |

### 이미지 업로드 방법

1. **파일 선택** 버튼 클릭 → 사진 파일 선택 (JPG, PNG 등)
2. **이미지 업로드** 버튼 클릭 → 업로드 완료 후 미리보기 확인
3. 모든 항목 입력 완료 후 **트레이너 등록** 버튼 클릭

> 이미지 업로드 전에는 등록 버튼이 활성화되지 않습니다.
> 이미 사용 중인 ID를 입력하면 오류가 표시됩니다.

## 6. 자주 묻는 질문 (FAQ)

**Q. 회원이 예약을 직접 취소할 수 있나요?**
A. 현재 회원 전용 취소 기능은 없습니다. 예약 취소는 관리자 페이지에서 직원이 처리해 주세요.

**Q. 같은 트레이너, 같은 날짜, 같은 시간에 두 명이 예약할 수 있나요?**
A. 불가능합니다. 이미 예약된 시간은 예약 화면에서 선택할 수 없도록 비활성화됩니다.

**Q. 예약 완료 화면에서 새로고침하면 정보가 사라집니다.**
A. 예약 완료 페이지는 예약 직후에만 표시되는 화면입니다. 예약 내역은 "내 예약" 메뉴에서 전화번호로 다시 조회할 수 있습니다.

**Q. 트레이너를 삭제하거나 수정하려면 어떻게 하나요?**
A. 현재 수정/삭제 기능은 개발사를 통해 처리해야 합니다. 필요 시 문의해 주세요.

**Q. 관리자 페이지에 비밀번호를 설정하고 싶습니다.**
A. 기본 제공 사양에는 포함되지 않습니다. 추가 개발이 필요하며 별도 문의 바랍니다.
