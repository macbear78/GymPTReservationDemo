# 숨고 서비스 등록 전체 문서

> 이 문서는 숨고에 웹사이트 제작 서비스를 등록하기 위한 모든 글을 포함합니다.
> 각 항목을 숨고 입력 폼에 그대로 복붙해서 사용하세요.

---

## ① 프로필 소개 (고수 소개란)

```
안녕하세요. 웹 풀스택 개발자입니다.

헬스장·PT샵·필라테스·요가 등 피트니스 업종 전문 웹사이트부터
일반 소상공인 홍보 사이트, 예약 시스템까지 직접 기획·디자인·개발을 모두 담당합니다.

외주 제작 경험을 바탕으로 클라이언트가 원하는 것을 빠르게 파악하고,
완성도 높은 결과물을 납기 내에 전달하는 것을 가장 중요하게 생각합니다.

▪ 사용 기술: Vue 3, React, Tailwind CSS, Node.js
▪ 특기: 모바일 반응형, 스크롤 애니메이션, 예약·회원 시스템
▪ 포트폴리오 데모: 실제 운영 수준의 PT 예약 시스템 완성본 보유
▪ 소통: 카카오톡·전화 빠른 응대, 수정 요청 3회 무료
```

---

## ② 서비스 제목

### 메인 타이틀 (최대 50자)
```
헬스·PT·필라테스 전문 홈페이지 제작 | 예약시스템·관리자 포함
```

### 부제목 / 태그라인 (있는 경우)
```
스크롤 애니메이션·모바일 최적화·관리자 대시보드까지 풀패키지
```

---

## ③ 서비스 카테고리 선택

```
IT·프로그래밍 > 웹사이트 제작·개발 > 홈페이지 제작
```

추가 태그 (숨고 태그 입력란):
```
홈페이지제작, 웹사이트제작, 헬스장홈페이지, 피트니스홈페이지, 예약시스템,
PT샵홈페이지, 필라테스홈페이지, 관리자페이지, 반응형웹, Vue, 랜딩페이지
```

✔ 구글 지도 연동
✔ 카카오톡 채널 / 전화 연결 버튼
✔ SEO 기본 설정 (페이지 타이틀, 메타 태그)
✔ 수정 3회 무료


────────────────────────────
■ 이런 분께 추천드립니다

• 기존 홈페이지가 오래되거나 모바일에서 깨지는 분
• 예약을 카카오톡으로만 받고 있어서 번거로우신 분
• 회원 관리를 엑셀로 하고 계신 분
• 깔끔하고 세련된 브랜드 이미지를 원하시는 분


────────────────────────────
■ 진행 방식
────────────────────────────

1단계 상담 — 원하시는 기능, 디자인 방향 파악 (무료)
2단계 기획 — 페이지 구성, 메뉴 구조 확정
3단계 디자인 — 시안 1~2개 제시 후 확정
4단계 개발 — 기능 개발 및 콘텐츠 입력
5단계 검수 — PC·모바일 최종 점검 및 수정
6단계 납품 — 도메인·호스팅 연결 또는 파일 전달


────────────────────────────
■ 자주 묻는 질문
────────────────────────────

Q. 도메인·호스팅은 포함인가요?
A. 기본 패키지에는 미포함입니다. 별도 구매 안내해 드리며,
   연간 약 3~10만 원 수준입니다. 대신 연결 설정은 제가 해드립니다.

Q. 완성 후 내용을 직접 수정할 수 있나요?
A. 네. 관리자 페이지에서 이미지, 텍스트, 트레이너 정보 등을
   코딩 없이 직접 수정하실 수 있습니다.

Q. 헬스장이 아닌 업종도 가능한가요?
A. 가능합니다. 필라테스, 요가, 크로스핏, 무용, 수영 등
   피트니스 관련 업종은 모두 대응 가능합니다.
   그 외 소상공인 홍보 사이트도 제작 가능합니다.

Q. 결제 기능도 넣을 수 있나요?
A. 네. 토스페이먼츠, 카카오페이 등 국내 PG사 연동 가능합니다.
   (결제 기능은 플러스 패키지 이상)

Q. 납기는 얼마나 걸리나요?
A. 기본 패키지 기준 약 2~3주, 풀패키지 기준 4~6주입니다.
   일정이 촉박한 경우 사전 협의 후 조정 가능합니다.


────────────────────────────
■ 유의사항
────────────────────────────

• 콘텐츠(사진, 텍스트, 로고)는 고객님이 제공해 주셔야 합니다.
• 무료 수정 3회 초과 시 건당 추가 비용이 발생합니다.
• 계약 후 작업 착수 시 취소·환불이 어려울 수 있습니다. 상담 충분히 후 진행해주세요.
• 데모 이미지는 저작권 무료 이미지(Unsplash)를 사용했으며, 실제 제작 시 고객님 사진으로 교체됩니다.
```

## ⑤ 패키지 구성 및 가격표

숨고에서 패키지를 3단계로 등록하는 것을 권장합니다.

---

### BASIC — 홍보용 홈페이지

**가격: 50만 원**
**납기: 2~3주**

```
✔ 메인 홈페이지 1페이지 (소개·프로그램·트레이너·찾아오시는길)
✔ 모바일 반응형
✔ 스크롤 애니메이션
✔ 상담 문의 폼
✔ 구글 지도 연동
✔ SEO 기본 설정
✔ 무료 수정 3회
✗ 예약 시스템 없음
✗ 관리자 페이지 없음
✗ 회원 로그인 없음
```

---

### STANDARD — 예약 시스템 포함

**가격: 150만 원**
**납기: 3~4주**

```
✔ BASIC 모든 항목 포함
✔ 트레이너 목록 / 상세 페이지
✔ PT 예약 시스템 (날짜·시간·트레이너 선택)
✔ 회원 로그인 / 회원가입
✔ 회원 대시보드 (내 예약 확인·취소)
✔ 관리자 대시보드 (예약 승인·취소·관리)
✔ 트레이너 등록·수정 (관리자)
✔ 매장 정보 직접 수정 (CMS)
✔ 무료 수정 3회
✗ 결제 기능 없음
✗ 패스 상품 판매 없음
```

---

### PREMIUM — 풀패키지 (결제·통계 포함)

**가격: 300만 원~**
**납기: 5~6주**

```
✔ STANDARD 모든 항목 포함
✔ PT 패스 상품 구매 페이지
✔ 결제 연동 (토스페이먼츠 / 카카오페이)
✔ 패스 관리 (상품 등록·수정·판매 현황)
✔ 매출·예약 통계 대시보드 (차트)
✔ 회원 관리 (가입 회원 목록·패스 현황)
✔ 상담 문의 관리 (관리자)
✔ 알림 시스템 (예약 확정·취소 알림)
✔ 관리자 알림 발송
✔ 무료 수정 5회
```

> 가격은 기능 범위에 따라 조정됩니다. 견적 문의 주시면 맞춤 견적을 드립니다.

GymPT — 헬스·PT샵 예약 시스템 풀패키지 데모

```
헬스장·PT샵을 위한 풀스택 예약 관리 시스템 데모입니다.

[ 구현 기능 ]

▸ 메인 랜딩페이지
  - 풀스크린 이미지 슬라이더 히어로 (4장 자동 전환, 진행 애니메이션 dot)
  - 스크롤 애니메이션 (fade-up, fade-left, zoom-in 등 6가지 효과)
  - 브랜드 소개, 운동 프로그램, 트레이너팀, 회원 스토리 캐러셀
  - 인스타그램 피드 섹션, 구글 지도 + 연락처
  - PC·모바일 완전 반응형

▸ 회원 서비스
  - 회원 로그인 / 회원가입
  - PT 패스 상품 선택 및 구매
  - 트레이너·날짜·시간 선택 예약
  - 내 예약 목록 확인 및 취소
  - 알림 센터 (예약 확정, 취소 등)

▸ 관리자 시스템
  - 관리자 전용 로그인 (JWT 인증)
  - 예약 목록 승인·취소 처리
  - 회원 목록 및 패스 현황
  - 트레이너 등록·수정·삭제
  - PT 패스 상품 관리
  - 매출·예약 통계 (차트)
  - 상담 문의 접수·처리
  - 알림 발송 관리
  - 매장 정보 CMS (이름, 주소, 전화, 히어로 이미지 등)

[ 사용 기술 ]
Vue 3, Vue Router, Tailwind CSS, Vite, JavaScript (ES2022)
IntersectionObserver 기반 커스텀 스크롤 애니메이션 Directive
SVG 도넛 차트 직접 구현

[ 디자인 특징 ]
- 6가지 메인 디자인 버전 보유 (A/B 테스트 또는 고객 선택 제공)
- 라임 포인트 컬러 + 다크 배경의 프리미엄 피트니스 무드
- Playfair Display + Pretendard 폰트 조합
- 다크모드 자동 대응
